# PhishingEmail
This is a proof of concept phishing email that looks very similar to an Apple Find My notification email. Uses only HTML to deceive users into clicking a button that redirects to a Grabify link.

HOW TO USE:
Replace iCloud link with your IP grabber link, then open html file in a broswer. Press crtl + a, ctrl + c and go to your preferred email and paste into a new email.
